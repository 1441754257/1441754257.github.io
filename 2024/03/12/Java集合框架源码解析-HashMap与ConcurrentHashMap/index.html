总结

HashMap和ConcurrentHashMap核心要点：

  1. JDK1.8使用数组+链表+红黑树结构
  2. 链表转红黑树阈值8，红黑树转链表阈值6
  3. 容量始终为2的幂，利用位运算提升效率
  4. ConcurrentHashMap使用CAS+synchronized保证线程安全
  5. JDK1.8采用尾插法避免环形链表问题
