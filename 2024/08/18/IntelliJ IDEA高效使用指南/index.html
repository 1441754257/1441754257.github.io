IntelliJ IDEA高效使用指南

IntelliJ IDEA高效使用指南

IDEA是Java开发者最常用的IDE，掌握快捷键和高级技巧能大幅提升开发效率。

一、必会快捷键

1.1 通用快捷键

快捷键 功能
Ctrl+Shift+F 全局搜索
Ctrl+Shift+R 全局替换
Ctrl+N 查找类
Ctrl+Shift+N 查找文件
Ctrl+Shift+A 查找动作
Shift+Shift 万能搜索

1.2 编辑快捷键

快捷键 功能
Ctrl+D 复制当前行
Ctrl+Y 删除当前行
Ctrl+Shift+↑/↓ 移动行
Ctrl+/ 注释/取消注释
Ctrl+Alt+L 格式化代码
Ctrl+Alt+O 优化导入
Ctrl+Shift+Enter 补全语句
Alt+Enter 快速修复

1.3 导航快捷键

快捷键 功能
Ctrl+B 跳转到定义
Ctrl+Alt+B 跳转到实现
Ctrl+U 跳转到父类
Ctrl+H 类层次结构
Ctrl+Alt+H 调用层次
Alt+↑/↓ 跳转方法
Ctrl+G 跳转到行

1.4 调试快捷键

快捷键 功能
F8 单步执行
F7 进入方法
Shift+F8 跳出方法
F9 恢复运行
Ctrl+F8 切换断点
Alt+F8 计算表达式

二、高效编码

2.1 Live Templates

2.2 Postfix Completion

2.3 多光标编辑

  • Alt+J：选中下一个相同词
  • Alt+Shift+J：取消选中
  • Ctrl+Alt+Shift+J：选中所有相同词
  • 鼠标中键拖动：列编辑模式

三、代码重构

3.1 重命名

  • Shift+F6：重命名变量/方法/类
  • 支持全局重命名，自动更新引用

3.2 提取方法

  • Ctrl+Alt+M：提取选中代码为方法

3.3 提取变量

  • Ctrl+Alt+V：提取为局部变量
  • Ctrl+Alt+F：提取为成员变量
  • Ctrl+Alt+C：提取为常量
  • Ctrl+Alt+P：提取为参数

3.4 内联

  • Ctrl+Alt+N：内联变量/方法

四、调试技巧

4.1 条件断点

4.2 表达式求值

4.3 热部署

  • 使用JRebel或Spring Boot DevTools
  • 修改代码后自动重新加载

五、插件推荐

插件 功能
Lombok 简化POJO
MyBatisX MyBatis增强
Translation 翻译
Key Promoter X 快捷键提示
Rainbow Brackets 彩色括号
GitToolBox Git增强
CamelCase 驼峰转换
JSON Parser JSON格式化

六、性能优化

6.1 内存设置

6.2 排除目录

6.3 禁用不需要的插件

总结

IDEA使用要点：

  1. 掌握常用快捷键
  2. 善用Live Templates
  3. 多用重构功能
  4. 调试技巧提升排查效率
  5. 安装必要插件
